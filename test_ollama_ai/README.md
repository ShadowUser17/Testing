#### Usage:
```bash
export OLLAMA_HOST="http://ollama.docker:11434"
```
```bash
export OLLAMA_MODEL="gemma3:1b-it-qat"
```
```bash
./env/bin/python3 <script> <context>
```

#### Show installed models:
```bash
curl "http://ollama.docker:11434/api/tags"
```
